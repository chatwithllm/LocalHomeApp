# LocalHomeApp

Privacy-first local grocery receipt intelligence system.

## Core principles
- Telegram is the primary intake channel for V1
- Raw receipts remain local to the machine/operator-controlled storage
- OCR is local-only
- GitHub stores non-sensitive code and documentation only
- Development follows documented phases and PRD-controlled scope

## Initial focus
- Project foundation
- Telegram intake architecture
- Dual local OCR strategy
- Structured receipt extraction
- Local storage and household intelligence

See `docs/PRD.md` and `docs/ROADMAP.md` for the source-of-truth plan.
