# Development Setup

## Purpose
Define how to set up LocalHomeApp for local development without committing sensitive data.

## Principles
- Code and docs live in Git.
- Real receipts, OCR artifacts, databases, and secrets stay outside Git-tracked paths.
- Configuration is environment-driven.

## Initial Setup Targets
- Python 3.11+
- virtual environment
- local `.env` file from `.env.example`
- configurable local data root outside the repository
